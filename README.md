# cpm

castis package manager.

(Yes, inspired by npm.)


## Usage

```
$ cpm help

Usage: cpm <command>
where <command> is one of: help, info, init, install, list, remove
cpm <cmd> -h    quick help on <cmd>

$ cpm init
$ cpm install ctail
$ cpm list
$ cpm remove ctail
...
```
